# 基于 Vue2 + element-ui 后台管理系统 Base

## 开始

```bash
# 安装(源为国外，网速较慢，如失败可删除node_modules重试)
npm install --registry=https://registry.npm.taobao.org

# 或 yarn 安装
yarn

# 启动
npm run dev
```

在浏览器打开: http://localhost:8585

## 构建

```bash
# 构建生产环境
npm run build
```

## 格式化

```bash
# 代码格式检查
npm run lint

# 代码格式检查和自动修复
npm run lint -- --fix
```
